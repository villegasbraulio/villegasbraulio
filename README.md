## Lo que aporto a un proyecto

- Visión de producto y capacidad para llevar una idea hasta una aplicación
  funcional end-to-end.
- Modelado de dominios complejos, APIs REST y procesos de negocio.
- Interfaces responsive con foco en claridad, accesibilidad y conversión.
- Integraciones con pagos, logística, servicios externos e inteligencia
  artificial.
- Tests automatizados, CI/CD, observabilidad y documentación técnica.
- Criterio para equilibrar velocidad de entrega, seguridad y mantenibilidad.
